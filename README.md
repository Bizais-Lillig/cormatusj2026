# Corm@USJ2026
An introduction to text extraction, structuration, and mining for the Université Saint-Joseph students, Spring 2026

Introduction à l'extraction, la structuration et la fouille de texte pour les étudiants de l'Université Saint-Joseph (Beyrouth) au printemps 2026

## Choix du corpus
1. Analyse du cadre juridique : https://www.irpi.fr/upload/pdf/etudes_juri/01_IRPI_Lanouvelle_loi_libanaise_sur_le_droit_d_auteur.pdf

2. Identification d'un auteur francophone : Charles Corm (1894-1963)

3. Deux textes :
- un recueil de poèmes : *La montagne inspirée* (https://gallica.bnf.fr/ark:/12148/bpt6k9106176d/f1.item)
- un roman : *Le volcan embrasé* (mis à disposition par la fondation Corm à Beyrouth)

Quelques extraits sont partagés dans cet espace afin de pouvoir utiliser les scripts (cf. dossier "DATA").

## Méthodes présentées
- moissonnage du web
- utilisation du manifeste IIIF
- extraction automatique du texte sur une image (automatic text recognition)
- extraction du texte à partir d'un fichier pdf
- encodage en XML-TEI
- requêtes XPath
- reconnaissance des entités nommées
- text clustering
- topic modeling
- salient features (TF-IDF)
- RAG (retrieval augmented generation)

## Remarques sur le code fourni
Afin de permettre aux étudiants suivant la formation d'utiliser les scripts concçus pour ce parcours sans installation particulière, ils sont fournis sous la forme de Jupyter notebooks utilisables directement sur Google Collab.

Cependant, pour ceux qui le souhaitent, les scripts peuvent être utilisés en local sur n'importe quel ordinateur, sauf mention contraire en tête de notebook. L'ensemble des librairies à installer dans son environnement en amont sont précisées en début de notebook.
